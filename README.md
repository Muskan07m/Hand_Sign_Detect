tap.py Flask App
Overview:
tap.py is a Flask app designed for hand sign detection using a webcam. It provides demo, data collection (triggered by pressing "s"), and model training functionalities.

Main Features:

Try Demo: Opens webcam for real-time gesture recognition.
Data Collection (S Button): Collects and stores hand gesture images when "s" key is pressed.
Train Button: Trains a model using collected data and saves as model.h5.
Usage:

Click "Try Demo" to use webcam for gesture recognition.
Press "S" to initiate and capture gesture images for data collection.
Click "Train" to train and save the model as model.h5.
Press "Esc" to stop the webcam.
Note: Ensure webcam accessibility for features requiring camera usage.
